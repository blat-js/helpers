# `getRandomInt`

```ts
import { getRandomInt } from "@blat-js/helpers/getRandomInt";

const rand1 = getRandomInt();
console.log(rand1); // Random integer between 0 and 10

const rand2 = getRandomInt(12345, 54321);
console.log(rand2); // Random integer between 12345 and 54321
```
